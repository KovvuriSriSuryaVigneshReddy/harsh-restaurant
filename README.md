# 🌿 Harsha Restaurant Website

A premium, modern, single-page multi-cuisine restaurant website built using clean HTML and CSS structure. Inspired by luxury tabbed restaurant menus, this responsive web interface features high-performance filtering, clean typography, dynamic customer components, and explicit action pathways optimized for mobile navigation.

## ✨ Features

### 🗂️ Interactive Menu Filtering
* **Smooth Category Switches:** Instantly switches between breakfast tiffins, appetizers/starters, main course curries, and rice dishes with lightning-fast pure vanilla JavaScript filtering logic.
* **Standardized Indicators:** Features industry-standard food markers—**Green Dot** icons for vegetarian dishes and **Brown Dot** icons for non-vegetarian dishes.
* **No-Price Display:** Highlights item titles and descriptions cleanly without upfront pricing lists, directing immediate focus to presentation.

### 📱 High-Conversion Action Controls
* **Strict UI Matching:** Integrated quick-access action options built from production mockups.
* **Call Now Button:** Highlights a solid, premium orange click-to-call standard button (`tel:` scheme implementation).
* **Glassmorphic Directions:** Includes a modern, semi-transparent backdrop-filtered mapping button linked straight to the Google Maps geolocation URL.

### 💬 Customer Feedback Component
* **Testimonial Cards Grid:** Includes stylized review quote boxes highlighting custom star counts and aesthetic block-quote notation styling.
* **Interactive Review Submission Form:** Built-in client-side functional feedback block containing an active star-selection array switcher and dynamic user response submission simulation alerts.
